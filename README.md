# SALAM DOT COM

A modern AI chat application built with React, TypeScript, and Tailwind CSS.

## 🌟 Features

- Real-time AI chat interface powered by OpenRouter
- Beautiful, responsive design with dark mode
- Modern animations and transitions using Framer Motion
- Type-safe development with TypeScript
- Sleek styling with Tailwind CSS

## 🚀 Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Duckizard/salam-dot-com.git
cd salam-dot-com
```

2. Install dependencies:
```bash
npm install
```

3. Create a `.env` file in the root directory and add your OpenRouter API key:
```
VITE_OPENROUTER_API_KEY=your_api_key_here
```

4. Start the development server:
```bash
npm run dev
```

The application will be available at `http://localhost:5173`

### Building for Production

To create a production build:

```bash
npm run build
```

The built files will be in the `dist` directory.
